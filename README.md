# 1º DAW - Curso 2025/2026

Este repositorio recopila los **ejercicios, prácticas y contenidos realizados**  durante el **primer curso del Ciclo Formativo de Grado Superior en Desarrollo de Aplicaciones Web (DAW)** en el curso **2025/2026**.
El objetivo es documentar mi aprendizaje y evolución en las distintas asignaturas del ciclo.

## Asignaturas

### [Programación]
Ejercicios principalmente en **Java**, trabajando:
- Fundamentos de Java
- Programación Orientada a objetos (POO)
- Estructuras de datos
- Clases, métodos y arrays

### [Lenguaje de Marcas]
Prácticas relacionadas con:
- HTML
- CSS
- XML
- JSON

### [Base de Datos]
Contiene ejercicios de:
- Diseño de bases de datos
- Modelo Entidad - Relación
- SQL (consultas, creación de tablas, usuarios...)

### [Diseño de interfaces Web] (asignatura de 2º año)
Prácticas sobre:
- Diseño Web
- Accesibilidad y usabilidad
- Maquetación Flex y Grid
- Prototipos
