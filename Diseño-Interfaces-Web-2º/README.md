# Diseño de Interfaces Web

Proyectos de diseño y maquetación web desarrollados durante la asignatura de Diseño de Interfaces Web de segundo año de DAW.

## Contenido
Tema 1. Planificación de una interfaz gráfica
   └── Elementos de diseño
   └── Color, tipografía e iconos
   └── Interacción persona-ordenador
   └── Interpretación de guías de estilo. Elementos
   └── Patrones de diseño Web
   └── Generación de documentos y sitios Web
   └── Componentes de una interfaz Web
   └── Aplicaciones para desarrollo Web
   └── Lenguajes de marcas
   └── Mapas de navegación. Prototipos
   └── Maquetación Web. Elementos de ordenación
   └── Plantillas de diseño

Tema 2. Accesibilidad
   └── Concepto de accesibilidad web. Barerras
   └── Consorcio World Wide Web (W3C)
   └── Pautas de Accesibilidad para el Contenido en la Web (WCAG)
   └── Niveles de orientación
   └── Niveles de prioridad y puntos de verificación
   └── Tabla de puntos de verificación
   └── Niveles de adecuación
   └── Herramientas de evaluación, análisis y testeo de accesibilidad web

Tema 3. Usabilidad
   └── Reglas y principios de usabilidad
   └── Inspiración para el diseño web
   └── Velocidad de carga de una aplicación web
   └── Herramientas de velocidad

Tema 4.1 Introducción

   └── Crear y vincular hojas d eestilo

      └── Ventajas de CSS Grid
      └── Primeros pasos
      └── Colocar los elementos en la rejilla
   └── Ejemplos de composiones de páginas
      └── Objetivo
      └── Composición de página con cajas flotantes
      └── Composición de página con flex

Tema 4.3 Creación de interfaces usando estilos. Diseño adaptativo
   └── Los Media Queries
   └── El tamaño d elas pantallas
   └── Ejemplo de sitio sencillo
   └── Imágenes adaptativas

Tema4.4 Tailwind
   └── Instalación y configuración
   └── Conceptos Base y Preflight
   └── Colores y personalización
   └── Tipografía
   └── Box-Model y Layout

   └── Pseudoclases y variantes

   └── Utilidades avanzadas
